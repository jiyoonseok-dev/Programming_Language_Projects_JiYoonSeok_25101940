🏰 ITM의 탑 (ITM Tower)
1. 프로그램 개요

ITM의 탑은 Python으로 제작된 텍스트 기반 턴제 RPG 게임이다.
플레이어는 ITM의 탑에 입장하여 총 4개의 층을 오르며, 각 층을 지키는 ITM 학생 보스를 물리쳐야 한다.
각 보스는 고유한 전투 패턴과 기믹을 가지고 있으며, 플레이어는 전략적인 선택과 확률 요소를 고려해 전투를 진행해야 한다.

본 프로그램은 조건문, 반복문, 함수, 클래스, 난수(random) 등을 활용하여 구현되었으며,
텍스트 기반 게임 설계와 확률적 시스템 이해를 목표로 한다.

2. 게임 진행 방식

- 게임은 1층 → 4층 순서로 진행된다.
- 각 층마다 보스와의 턴제 전투가 진행된다.
- 플레이어와 보스는 번갈아 가며 공격한다.
- 플레이어의 체력이 0 이하가 되면 GAME OVER가 되며, 처음부터 다시 도전할 수 있다.
- 각 보스 처치 후에는 체력 회복 또는 보상 선택이 주어진다.

3. 플레이어 시스템

3.1 플레이어 기본 능력치
   
초기 체력: 50
최대 체력: 50
초기 공격력: 1
상태 생존 여부 판단 가능

3.2 플레이어 행동

매 턴 공격 부위 선택: 머리 / 몸통 / 왼팔 / 오른팔 / 왼다리 / 오른다리
*일부 보스는 특정 부위 공격 시 특수 효과가 발생한다.

4. 층별 보스 설계

1층: 김현우 (튜토리얼 보스): 튜토리얼 층으로, 탑의 시스템과 전투 방식을 설명하는 역할을 한다.
체력: 5 / 공격력: 1 ~ 5 (랜덤)
특징: 머리 공격 시 크리티컬 발생 (3 데미지) / 기타 부위는 1 데미지 / 머리 크리티컬 시 → “김현우의 안경을 부쉈다!” 출력
보상: 김현우의 자취방 식칼 획득 -> 플레이어 공격력 5로 상승

<img width="356" height="567" alt="image" src="https://github.com/user-attachments/assets/237621f2-788f-4774-897e-946ed1685fe3" />

2층 : 고진형: 퍼즐형 보스로, 공격 부위 선택의 중요성을 강조한다.
체력: 1000 (보상에 따라 감소 가능) / 공격력: 10
특징: 모든 부위 공격 시 1 데미지만 적용 / 단, 오른다리 공격 시 즉사
특수 문구: “고진형은 오른쪽 십자인대가 없습니다! 크리티컬 x100!”
<img width="405" height="263" alt="image" src="https://github.com/user-attachments/assets/2a0d9634-efa1-4320-91e4-d5bd9234aa24" />

3층 : 최준희: 확률 요소와 리스크 관리가 핵심인 전투
체력: 30 / 공격력: 20
특징: 등장 대사: “SDT를 무시하지마라.” / 공격 시 50% 확률로 실패 -> 실패 시: “최준희가 어리버리하다!” / 몸통 공격 시 2배 데미지 -> “그의 심장은 여립니다..” / 공격 성공 시 대사: “가시고~”
<img width="394" height="515" alt="image" src="https://github.com/user-attachments/assets/ccbb2dee-c31b-49fc-a88d-0f0ea0a82900" />

4층 : 최홍서 (최종 보스): 운과 긴장감을 극대화한 최종 보스 전투
초기 체력: 1 / 초기 공격력: 1
특징: 등장 대사: “I'm spiderman” / 플레이어는 매 턴 확률적으로만 공격 성공 -> 턴이 지날 때마다: 공격력 증가, 회피율 증가 / 회피 시 대사: “스파이더맨 장 담모한다” / 첫 턴 공격 성공 시: “벌레 장 담모한다”

5. 보상 시스템

각 층 클리어 후 다음 중 하나를 선택할 수 있다.

- 체력 완전 회복
- 김우제 교수님의 BIS A+: 공격력 +5
- 권혁윤 교수님의 장학금: 최대 체력 +10
- 프로그래밍 별점 테러: 이후 등장하는 적들의 체력 -10
